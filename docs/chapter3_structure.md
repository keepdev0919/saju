# 🏛️ [제 3서: 천개의 비밀 - 7장 구성안]

전통 분류를 천명록 브랜드에 맞게 고급화한 네이밍입니다.

---

| 장 | 전통 분류 | 천명록 네이밍 | 부제 | 라이브러리 데이터 |
| :---: | :---: | :---: | :---: | :---: |
| **1장** | 성정(性情) | **본성(本性)** | 근원의 불꽃 | 일간 + 오행 균형 |
| **2장** | 재물(財物) | **재록(財祿)** | 부의 그릇 | 재성(편재/정재) |
| **3장** | 관록(官祿) | **관운(官運)** | 천직의 자리 | 관성(정관/편관) |
| **4장** | 혼인(婚姻) | **연분(緣分)** | 인연의 실타래 | 성별 맞춤 십성 + 도화살 |
| **5장** | 건강(健康) | **체상(體象)** | 신체의 등불 | 오행 불균형 |
| **6장** | 대운/세운 | **시운(時運)** | 시간의 파도 | 대운 + 향후 5년 세운 |
| **7장** | 귀인/신살 | **비책(秘策)** | 개운의 열쇠 | 귀인 + 기신 + 보완 방향 |
