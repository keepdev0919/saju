# 📜 만세력 라이브러리 데이터 명세서 (Client Meeting Spec Sheet)

> **문서 목적**: 클라이언트 미팅 시, 현재 구현된 기능과 라이브러리를 통해 **추가로 확장 가능한 기능**을 명확히 구분하여 제안하기 위함입니다.
> **기반 라이브러리**: `lunar-javascript` (정밀 만세력 엔진)

---

## ✅ 1. 현재 구현 및 사용 중인 데이터 (Current Implementation)
현재 시스템이 AI에게 제공하여 분석하고 있는 기초 데이터입니다.

| 데이터 항목 | 설명 | 용도 |
| :--- | :--- | :--- |
| **사주팔자 (Four Pillars)** | 년/월/일/시의 간지 (예: 갑자년, 병인월...) | 사람의 고유한 DNA 코드 (기본 분석 자료) |
| **일간 (Day Master)** | 태어난 날의 천간 (예: 갑목, 병화) | 사용자의 **본질적인 성향** ("당신은 큰 산입니다") 분석 |
| **오행 분포 (Five Elements)** | 목, 화, 토, 금, 수의 비율 및 점수 | **건강운, 기질 분석** ("화 기운이 강해 성격이 급함") |
| **용신 (Yong-shen)** | 사주의 균형을 맞추는 핵심 오행 | **행운의 요소** (행운의 색, 숫자, 아이템) 추천 |
| **성별/생년월일** | 기본 인적 사항 | 대운/세운 계산의 기초 (남녀 순행/역행 차이) |

## 🔓 2. 라이브러리로 추출 가능한 데이터 (Available Capabilities)
현재 코드로 구현되어 있지는 않으나, **라이브러리 자체 기능으로 즉시 뽑아낼 수 있는 데이터**입니다.
클라이언트가 **특정 컨셉(재물 특화, 연애 특화, 올해의 운세 등)**을 요구할 때 아래 "메뉴"를 골라달라고 하면 됩니다.

### A. 운세의 흐름 (Timeline Analysis) - *가장 중요*
> "내년에는 돈을 벌 수 있을까?" 같은 질문에 답하기 위해 필수적입니다.

*   **대운 (Da-un, 10-year Cycle)**: 10년마다 바뀌는 인생의 큰 계절 (예: "34세~43세는 재물운이 들어오는 시기")
*   **세운 (Se-un, Yearly Luck)**: 매년 들어오는 운 (예: "2026년 병오년은 이동수가 강함")
*   **월운 (Wol-un, Monthly Luck)**: 매월 들어오는 운 (예: "3월은 조심해야 함")

### B. 성격 및 사회성 분석 (Ten Gods / Sip-sin) - *심리 분석에 탁월*
> "이 사람이 사업가 체질인가? 직장인 체질인가?"를 판단합니다.

*   **십신 (Ten Gods)**: 일간(나)과 다른 글자와의 관계
    *   **비견/겁재**: 경쟁심, 자존심, 형제/동료
    *   **식신/상관**: 표현력, 창의성, 말솜씨 ("유튜버 해도 될까요?")
    *   **편재/정재**: 재물, 결과, 아버지/아내 ("사업운이 있나요?")
    *   **편관/정관**: 명예, 직장, 규율, 남편 ("공무원 시험 합격할까요?")
    *   **편인/정인**: 공부, 문서, 어머니, 인내심 ("부동산 문서운")

### C. 기운의 강약 (12 Phases) - *상태 분석*
> "운이 들어왔는데, 내가 그걸 감당할 힘이 있는가?"를 봅니다.

*   **12운성 (12 Phases of Qi)**: 인간의 생로병사에 빗댄 12단계 에너지
    *   **장생/제왕**: 에너지가 가장 왕성함 (사회적 성공 유리)
    *   **쇠/병/사/묘**: 에너지가 약해짐 (내실을 다져야 함, 건강 주의)

### D. 특수 살 (Divine Spirits / Sin-sal) - *흥미 유발 요소*
> 일반 대중들이 가장 흥미로워하는 "도화살", "역마살" 같은 키워드입니다.

*   **도화살 (Peach Blossom)**: 인기, 매력, 연예인 기질
*   **역마살 (Travel Horse)**: 이동, 변동, 해외 운
*   **화개살**: 예술, 종교, 고독
*   **귀인 (Nobleman)**: 나를 도와주는 귀한 존재 (천을귀인 등)
*   **공망 (Void)**: 구멍 난 독. 노력해도 결과가 잘 안 나오는 영역.

## 🤖 3. AI 프롬프트 확장 예시 (Proposal)
클라이언트가 위에서 기능을 선택하면, 아래와 같이 질문을 던져서 답변을 생성할 수 있습니다.

**[예시: 2026년 재물운 특화 요청 시]**
> **입력 데이터**: 기존 사주 + **대운(편재)** + **2026년 세운(식신)** + **12운성(제왕)**
>
> **AI 프롬프트**:
> "이 사용자는 타고난 재물 그릇은 작지만,
> **현재 10년 대운에서 큰 재물(편재)**이 들어왔고,
> **2026년에는 재주를 부려 돈을 만드는(식신)** 운이 **가장 강력한 힘(제왕)**으로 들어옵니다.
> 따라서 2026년은 무조건 사업을 확장해야 하는 시기라고 분석해 주세요."

## 📝 4. 결론 (Summary)
*   **현재**: "당신은 누구인가?" (기본 성향) 위주의 분석
*   **확장 가능**: "그래서 언제 대박 나는가?" (시기, 구체적 사건) 분석 가능

## 💻 5. 라이브러리 출력 데이터 예시 (Raw JSON Code)
클라이언트 미팅 시 **"실제로 이런 데이터가 나옵니다"**라고 보여줄 수 있는 예시 코드입니다.
(현재 시스템이 이 모든 것을 다 계산할 수 있음을 증명하는 자료)

### 🔹 기초 데이터 (Basic)
```json
{
  "bazi": {
    "year": "갑자(甲子)",
    "month": "병인(丙寅)",
    "day": "무진(戊辰)",
    "hour": "임술(壬戌)"
  },
  "dayMaster": "무토(큰 산)",
  "wuxing": { 
    "wood": 30, "fire": 40, "earth": 20, "metal": 0, "water": 10 
  }
}
```

### 🔸 고급 분석 데이터 (Advanced - 기능 해제 시 즉시 확보 가능)
```json
{
  "daun": [ // 대운 (10년마다 바뀌는 운)
    { "age": 4, "ganZhi": "정묘", "tenGod": "정인", "energy": "제왕" },
    { "age": 14, "ganZhi": "무진", "tenGod": "비견", "energy": "쇠" },
    { "age": 24, "ganZhi": "기사", "tenGod": "겁재", "energy": "병" }
  ],
  "sipsin": { // 십신 (심리/관계 분석)
    "year": "편관",
    "month": "편인",
    "hour": "편재" // -> "말년에 큰 돈을 만짐" 해석 가능
  },
  "phases": { // 12운성 (에너지 강도)
    "year": "목욕", // -> "어린 시절 멋부리기 좋아함"
    "day": "관대"   // -> "중년에는 벼슬길에 오름"
  },
  "sinsal": [ // 신살 (특수 별)
    "도화살", "역마살", "천을귀인" // -> "인기가 많고 이동이 잦지만 귀인이 도움"
  ]
}
```
