# 📋 사주풀이 웹 플랫폼 - 프로젝트 요구사항

---

## 🎯 프로젝트 개요

**사주풀이 웹 플랫폼 개발 및 결제/인증 시스템 구축**

### 배경 및 목표
- 복잡한 회원가입 절차 없이, **결제 후 즉시 사주풀이 결과 확인** 가능한 웹 서비스
- **모바일 사용성 최우선** (PC에서도 모바일 비율 UI 제공)
- 카카오톡 알림톡을 통한 결과 링크 전송
- PDF 추가 구매(다운로드) 기능으로 수익 모델 다각화

### 핵심 컨셉
> **'로그인 없는 비회원 결제'** + **'카카오톡 링크를 통한 재진입'**

## 🏗️ 기술 스택

### Frontend
```
- React 18
- Vite
- Tailwind CSS
- lucide-react (아이콘)
```

### Backend (예정)
```
- Node.js / Express
- MySQL
- AWS (EC2, RDS, S3)
```

### 외부 연동 (예정)
```
- 사주 API (만세력 API 등)
- PG사: 포트원(아임포트)
- 카카오 알림톡 API
- PDF 생성: html2pdf.js 또는 Puppeteer
```
