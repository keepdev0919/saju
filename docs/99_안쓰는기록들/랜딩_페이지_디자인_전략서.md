# 랜딩 페이지 배경 전략: "고요한 새벽 (The Silent Dawn)"

## 디자인 비전: 프리미엄의 절제와 천상의 신비
랜딩 페이지 배경은 단순한 바탕이 아니라, 사용자가 깊은 영적 경험을 준비하도록 안내하는 **포털(Portal)**의 역할을 수행해야 합니다. **'절제(Restraint)'**와 **'럭셔리한 물성'**의 원칙에 따라, 수묵화(Sumi-e) 미학을 중심으로 한 디자인을 제안합니다.

---

## 1. 시각적 구성: "여백의 미"
복잡한 사진 대신 사용자와 함께 호흡하는 **디지털 수묵화**를 사용합니다.

*   **수묵 그라데이션**: 상단의 **심연의 검정 (#050505)**에서 하단의 **옅은 숯색 (#1A1A1C)**으로 흐르는 수직 그라데이션.
*   **안개 낀 지평선**: 화면 하단 30% 영역에 신비로운 안개에 쌓인 **고대 산맥**의 실루엣 (한국 산수화의 느낌)을 배치.
*   **중앙의 명료함**: 입력 폼이 위치하는 화면 중앙은 미니멀하게 유지하여 사용자가 데이터 입력에 온전히 집중할 수 있도록 함.

## 2. 물성과 질감 (Materiality)
*   **4K 한지 텍스트**: 고해상도 **전통 한지** 스캔 데이터를 낮은 투명도(약 5-8%)로 배경에 오버레이. 이는 평면적인 디지털 UI가 아닌 실물 '고문서'를 만지는 듯한 촉각적 경험을 제공함.
*   **섬유질의 디테일**: 자세히 보았을 때 나타나는 섬유 질감을 통해 '장인 정신'이 깃든 고품격 인상을 심어줌.

## 3. 동적 요소 (Micro-interactions)
*   **떠다니는 불꽃/꽃잎**: 부적 테마와 일치하는 단 하나의 매우 정교한 **꽃잎**이나 **황금빛 불꽃**이 패럴랙스 효과와 함께 화면을 가로질러 천천히 이동함.
*   **안개의 호흡**: 산맥 주변의 안개가 아주 천천히(15-20초 주기) 투명도를 바꾸며 맥동하게 하여 대자연의 호흡을 시뮬레이션함.
*   **붓터치 포컬 포인트**: 화면의 일부를 수직으로 나누는 굵고 강렬한 **먹선**을 통해 '두루마리가 펼쳐지는' 느낌을 시각화함.

## 5. 디자인 회의 결론
> [!IMPORTANT]
> **결론**: 배경은 **"무겁지만 숨을 쉬는 듯한"** 느낌이어야 합니다. 5,000년 전통의 무게감을 담으면서도 기술적으로는 현대적이고 반응성이 좋아야 합니다. 실사 사진을 배제하고 수묵의 추상성을 수용함으로써, [천명록]을 일반적인 운세 앱과 차별화하는 '하이엔드 브랜드' 정체성을 유지합니다.
