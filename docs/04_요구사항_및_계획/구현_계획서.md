# 구현_계획서: 프리미엄 랜딩 페이지 배경 - "고요한 새벽"

## 목표
부적 작품과 어울리는 하이엔드 절제미를 갖춘 한국적 에스테틱의 "고요한 새벽" 랜딩 페이지 배경을 구현함.

## 제안된 변경 사항

### 1. 에셋: 이미지 생성
- [신규] `public/images/landing/mountain_ink.png`: 투명 배경을 가진 가로형 수묵화 산맥 실루엣 생성.

### 2. 스타일: 유틸리티 확장
#### [수정] `src/index.css`
- 느리게 불투명도가 변하는 `@keyframes mist-breathing` 추가.
- 한지 텍스처를 위한 개선된 블렌딩 모드를 갖춘 `.bg-hanji-premium` 정의.
- 재(ash)에서 먹(ink)으로 변하는 수직 전이를 위한 `.ink-gradient` 추가.

### 3. 프론트엔드: 랜딩 페이지 고도화
#### [수정] `src/components/SajuApp.jsx`
- 기존의 인라인 배경 그라데이션을 구조화된 `Background` 레이어로 교체.
- 초기 로드시 "천상의 두루마리"가 펼쳐지는 애니메이션 구현.
- `mountain_ink.png` 에셋을 하단 고정 영역에 패럴랙스 스크롤 효과와 함께 추가.

## 검증 계획
### 자동화 테스트
- 해당 없음

### 수동 검증
- 배경 요소와 입력 필드 간의 시각적 균형 확인.
- 안개 및 호흡 애니메이션의 부드러움 확인.
- 한지 텍스처가 방해되지 않으면서도 물성을 더할 만큼 충분히 은은한지 확인.
